# Time Series-Econometrics (ECON 722)
This is the repository for Elena Pesavento's ECON 722 Time Series Econometrics Course at Emory University. 

The repository has a folder for each major topic/module that the class will go through. Be aware that the example datasets and the codes may vary in type. As such, you may be using one language for one part and another one for another.
